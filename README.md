## Usage

### Web Version

1. Clone project code.

```bash
git clone git@github.com:JimmyLzy/lifeRestart.git my-project
cd my-project
```

2. Installation dependence.

```bash
yarn install
```

Or

```bash
npm install
```

3. Start local server.

```bash
yarn dev
```

Or

```bash
npm run dev
```

4. After the startup is complete, will automatically open a browser and visit [http://localhost:8081/view/index.html](http://localhost:8081/view/index.html).

### Command Line Version

```bash
node repl
```
